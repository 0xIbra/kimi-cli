WebSearch tool allows you to search on the internet to get latest information, including news, documents, release notes, blog posts, papers, etc.

When searching for current events, live prices, or recent news, always set time_range to 'qdr:d' (past day) or 'qdr:h' (past hour) to get fresh results. For recent developments or weekly summaries, use 'qdr:w' (past week).

Use sources=['news'] when the user asks about breaking news or market events. Use country='RU' when the user writes in Russian. Set location for region-specific queries.
